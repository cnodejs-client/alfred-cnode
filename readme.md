## alfred-cnode

alfred workflow for cnodejs.org

### screenshot

![screenshot](https://raw.githubusercontent.com/nswbmw/alfred-cnode/master/screenshot.gif)

### Usage

```
cnode  -> 所有类型帖子
cnode good -> 精华帖子
cnode share -> 共享帖子
cnode ask -> 问答帖子
cnode job -> 招聘帖子
```
